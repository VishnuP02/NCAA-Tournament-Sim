# Data notes

`teams_2024_synthetic.csv` is a synthetic fallback dataset for the 2024 NCAA men's tournament field. It uses Torvik-style adjusted offensive efficiency, adjusted defensive efficiency, and tempo columns, but the numeric ratings are hand-constructed for portfolio/demo purposes rather than copied from a paid or scraped source.

The project is designed so a future user can replace this CSV with an allowed public ratings export that has the same columns.
